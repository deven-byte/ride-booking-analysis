# 🚖 Ride Booking Analytics Dashboard

## 📊 Project Overview
This project analyzes ride booking data to identify trends in revenue, demand, and driver performance using SQL and Power BI.

## 🛠️ Tools Used
- Excel (Data Cleaning)
- MySQL (Data Analysis)
- Power BI (Dashboard)

## 📂 Dataset
- 2500+ ride booking records
- Includes customers, drivers, vehicles, and bookings data

## 📈 Key Insights
- Peak ride demand occurs between 17:00–22:00
- Mumbai shows highest booking volume
- Top drivers contribute significant portion of trips
- UPI dominates payment methods
